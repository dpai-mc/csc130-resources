Information Needed Before Starting

Before you begin, please make sure you have the following credentials. You will receive these from your instructor.

  • OpenVPN Username
  • OpenVPN Password
  • Private Key Password: abnormal
  • WordPress IP Address (Your unique URL)
  • WordPress Username: admin
  • WordPress Password (this will be unique to you)

Setup Instructions

1

Download OpenVPN Connect

Download and install the OpenVPN Connect client for your operating system (Windows, macOS, or Linux).

Download OpenVPN
2

Download VPN Configuration Files

Click the link below to download the VPN configuration files. The zip file contains:

  • csc130-OpenVPN.ovpn (for off-campus access)
  • csc130-OpenVPN-onCampus.ovpn (for on-campus access)
  • secret (your credential file)
Download Configuration Files

Crucial: After unzipping, you must keep the secret file in the exact same folder as the .ovpn file you intend to use.

3

Add Your Credentials to the 'secret' File

You will be provided with a unique username and password. Open the secret file with a plain text editor (like Notepad on Windows or TextEdit on Mac).

  • On the very first line, enter your username.
  • On the second line, enter your password.

Important: There should be no extra lines or spaces. The file should only contain these two lines.

4

Import Profile and Connect

First, import the profile that matches your primary location (where you'll work most often).

  • If you are mostly off-campus, import csc130-OpenVPN.ovpn.
  • If you are mostly on-campus, import csc130-OpenVPN-onCampus.ovpn.

After importing, follow these connection steps:

  1. Verify Username: The app should display your correct username. This confirms it has found and read your secret file.
    • If it does not show your username, double-check that your .ovpn and secret files are in the same folder and that the content of the secret file is correct.
  2. Save Password: Check the box labeled "Save Private Key Password".
  3. Enter Password: When prompted, enter the private key password: abnormal.
  4. Connect: Click to connect. If you anticipate working from both locations, we recommend importing the second profile now as well. Follow the same steps to save the private key password so you're prepared to connect from anywhere.
5

Access Your WordPress Site

Once you are connected to the VPN, you can access your personal WordPress container. Your site is only accessible while the VPN is active.

  1. Visit your site: Open a web browser and enter your unique URL, which will be in the format: http://10.255.254.1xx (where xx is your student number).
  2. Log into the Dashboard: To edit your site, open a new browser tab and go to your admin dashboard by adding /wp-admin to your URL. For example: http://10.255.254.1xx/wp-admin.
  3. Use your credentials: Log in with the username admin and the unique, complex password provided by your instructor.

Password Warning: The WordPress password provided to you is complex and cannot be changed. You must save it in a secure location, such as your browser's password manager or a private file, so you do not lose it.
